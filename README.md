## 📖 Overview

The **TSUT (Text Summarization Using Transformers)** project explores abstractive text summarization by comparing two state-of-the-art transformer architectures. Specifically, we evaluate how a massive 1.6 Billion parameter **BART-large-CNN** (pretrained) performs against a lightweight 60 Million parameter **T5-small**, fine-tuned exclusively for 3 epochs.

To visualize our findings, training analytics, qualitative examples, and live text summarization capabilities, we built a **Premium Pro Max Dashboard** using React, Vite, and the HuggingFace Inference API.

## ✨ Features

- **📊 Comprehensive ROUGE Evaluation:** Detailed breakdown of ROUGE-1, ROUGE-2, and ROUGE-L scores evaluated on 100 benchmark CNN/DailyMail test articles.
- **📈 Training Analytics:** Interactive Loss curve charts mapping the convergence of the T5 model across its 3-epoch fine-tuning stage.
- **💬 Real-Time AI Summarization (Live Demo):** Paste any text to test how BART and T5 summarize it dynamically right in the browser via HuggingFace's API. Provides an instant purely extractive fallback if APIs are saturated.
- **🧭 Qualitative Examples:** Compare reference "Ground Truth" summaries directly alongside outputs generated by BART and T5.
- **🎨 Modern Web Architecture:** Built using an optimized React + Vite + TypeScript frontend adorned with a premium glassmorphism light theme.

## 📊 Findings & Key Insights

| Metric      | BART-large-CNN | T5-small (FT) | Δ Difference |
|-------------|----------------|---------------|--------------|
| **ROUGE-1** | 36.84          | 31.49         | + 5.35     |
| **ROUGE-2** | 16.44          | 12.46         | + 3.98     |
| **ROUGE-L** | 27.65          | 23.48         | + 4.17     |

**The Takeaway:** Despite T5-small having approximately **1/26th of the parameters** of BART, a quick 3-epoch fine-tuning allowed it to achieve **82% of BART's ROUGE-L score**. This highlights the sheer efficiency of parameter-efficient small language models when fine-tuned on task-specific domains rather than relying on generalized massive architectures.

## 💻 Running Locally

To run the TSUT Dashboard gracefully on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hardik-Sonawane/tsut_new.git
   cd tsut_new/frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```

4. **View locally:** Open your browser to `http://localhost:5173`.
